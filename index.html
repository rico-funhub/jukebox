STEP 3

Connect to Spotify

You'll log in on Spotify's page, then come back to the jukebox.

  1. On any computer, go to developer.spotify.com/dashboard and log in with the location's Spotify account.
  2. Click Create app. Name it anything (e.g. "FUNHUB Jukebox — [location]").
  3. For Redirect URI, enter exactly:
    https://rico-funhub.github.io/jukebox
  4. Under APIs, tick Web API and Web Playback SDK. Save.
  5. Open the app's Settings — copy the Client ID and paste it above.

The Client ID is not a password — it's a public identifier that's meant to be shared, and there's no secret to protect in a browser app. The real login (email & password) is entered on Spotify's own page, never here. For in-app playback the account must be Spotify Premium.
